Procedure-Map

2023-03-02

This tech note derive from the tutorial provided by the Youtube channel HeartBeast.

In procedure map, the map will be generated automatically and randomly during the playing and loading. It follows a specific rule. Therefore, players can enjoy the infinity area without any limitation theoretically.

Bitmask

After the geomap is loaded, click add new autotile and select the icon, then select bitmask as the illustration demonstrated as followed.

The red parts of bitmask are actually the conjunction of different block. Thus the blocks will automatically generated and turn to the suitable shape to fit.

Tips - Debug

When you instantiate one class like below

var walker = Walker.new(Vector2(19, 10), borders)

if “W” in “Walker.new” is in lowercase, it will refer to the variable “walker” instead of the class “walker” which is created earlier.

To fix that you can either change the class_name in class or change the variable name. Be sure the var name is different from class_name.

tip Why use func Ceil()
var top_left_corner = ( position - room_size/2 ).ceil()
Here I provide a sample that why we use func ceil() to locate the top left corner when the length and width are odd number.

  • position (3, 2): the room center
  • top_left (0, 0)
  • size (5, 3): length * width
  • 1/2 size (2.5, 1.5)

The if borders.has_point(room_floor_step): make the room floor steps fall inside border.
