shader-rotation

2024-09-13

目标，使用 gdshader 让 shader 的2D平面旋转起来
Let 2D sprite rotate and pixelate by GDshader.

The example code as above.

Rotation theory as below. It is about how coord rotate work.

https://www.youtube.com/watch?v=Ta8cKqltPfU

  1. shader_type canvas_item
    说明: 指定 Shader 类型为 canvas_item，适用于 2D 图形，如 Sprite 和 TextureRect。
    作用: 允许对 2D 渲染对象进行着色。

  2. render_mode unshaded;
    说明: 使用 unshaded 模式表示不进行光照计算，即不受光源影响。
    作用: 使得 Shader 只关注纹理的颜色和透明度，不考虑光照效果。

  3. uniform float speed = 1.0;
    说明: 定义一个 uniform 变量 speed，用于控制纹理旋转的速度。
    作用: 允许在编辑器中动态调整纹理旋转的速度。

  4. uniform float pixel_size : hint_range(1.0, 100.0) = 10.0;
    说明: 定义一个 uniform 变量 pixel_size，用于控制像素化效果的大小。hint_range 提供一个滑动条以调整值的范围。
    作用: 控制像素块的大小，实现像素化效果。值越大，像素块越大。

  5. vec2 rotateUV(vec2 uv, vec2 pivot, float rotation)
    说明: 定义一个函数 rotateUV 用于旋转 UV 坐标。
    输入参数:
    uv: 需要旋转的 UV 坐标。
    pivot: 旋转的中心点。
    rotation: 旋转角度。
    返回: 旋转后的 UV 坐标。

  6. void fragment()
    说明: fragment 函数是每个像素的处理逻辑。
    操作:
    获取当前 UV 坐标: vec2 uv = UV; 读取纹理的 UV 坐标。
    像素化 UV 坐标: uv = floor(uv * pixel_size) / pixel_size; 将 UV 坐标对齐到像素网格，创建像素化效果。
    计算旋转角度: float angle = TIME * speed; 根据时间动态计算旋转角度，使得纹理旋转。
    旋转 UV 坐标: vec2 rotated_uv = rotateUV(uv, center, angle); 对像素化后的 UV 坐标进行旋转。
    限制 UV 范围: rotated_uv = clamp(rotated_uv, vec2(0.0), vec2(1.0)); 确保 UV 坐标在纹理边界内，防止边缘拉伸。
    采样纹理: vec4 tex_color = texture(TEXTURE, rotated_uv); 从纹理中采样颜色和透明度。
    输出颜色: COLOR = tex_color; 将采样到的颜色和透明度输出为最终颜色。
    总结
    目标: 实现一个旋转且像素化的纹理，同时确保透明区域保持透明。
    旋转处理: 使用旋转矩阵对 UV 坐标进行旋转。
    像素化效果: 通过对 UV 坐标进行像素化处理，实现像素块效果。
    透明度处理: 保持纹理的透明区域不受影响，确保最终渲染结果正确显示透明部分。
