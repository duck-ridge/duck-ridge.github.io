This is a tech note about how to create an RTS-like selection box in Godot 4.2

Actually, it is been a while since Godot 4 released. There are some reasons stoped me to try it.

  1. I was developing my game Effluent in 3.5 already and numerous rules and functions changed.
  2. New Engines always have some inherited problems.

Initially, I decide to try C#. However, when I trying to make it quick, the easy use of GDscript always make me stay.

I am working on selection these days and it looks OK.

I will first introduce these variable
is_dragging: if player entering a selection mode
drag_start_pos: the start position of ColorRect
drag_end_pos: the end position of ColorRect

Logic of drag selection Rect
Set is_dragging true when pressed LMB.
The “if is_dragging != true:” can save some energy, in case of repeated update drag_start_pos

Update the mouse position all the time
when the mouse position is on the Fourth Quadrant relative to the drag_start_pos. Set the scale negative.

TBH, this code chuck is quite redundant, I do some improvement to it

When press ended we will know which points are in the rect selected.
