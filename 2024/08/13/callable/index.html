callable

2024-08-13

最近在适应Godot4，有很多东西都发生了变化，其中signal/connect也换成了一个全新的格式
Recently, I have been adapting to Godot 4, and many things have changed, including the signal/connect system, which has been replaced with a new format.

新的信号连接格式为 connect(signal_name, callable, flags)
The new signal connection format is: connect(signal_name, callable, flags)

Callable
Callable 是Godot 4中引入的一个新概念，用于封装可以被调用的方法或函数。你可以将 Callable 看作是一个方法的“指针”或引用，它使得方法可以在未来被调用。其主要目的是使得信号连接和回调变得更加灵活和清晰。

Callable is a new concept introduced in Godot 4 that encapsulates methods or functions that can be invoked. You can think of a Callable as a “pointer” or reference to a method, allowing the method to be called at a later time. The main purpose of Callable is to make signal connections and callbacks more flexible and clear.

通过 Callable(self, “my_function”)，你可以创建一个指向 my_function 方法的 Callable 实例。
By using Callable(self, “my_function”), you create a Callable instance that points to the my_function method.

你可以通过 my_callable.call() 来执行该方法，就像直接调用 my_function() 一样。而 Callable.bind() 方法可以将参数绑定到 Callable，以便在未来调用时自动传递这些参数。
You can execute the method using my_callable.call(), just as if you were calling my_function() directly. The Callable.bind() method allows you to bind parameters to the Callable, so they will be automatically passed when called later.
